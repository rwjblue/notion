[−][src]Crate node_archive

This crate provides types for fetching and unpacking Node distribution archives, which is a tarball for Unixes and a zipfile for Windows.

These docs show the top-level exports of this crate as re-exported of the tarball module (due to limitations of rustdoc); the top-level exports are re-exported from tarball on Unix operating systems and from zip on Windows operating systems.
